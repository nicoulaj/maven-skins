1024px skin

The 1024px skin is a skin for Maven generated sites. It is derived from the 1024px template by Andreas Viklund.

The site you are currently seeing is generated using this skin.

Usage

To use maven-1024px-skin as your Maven generated site skin, declare it in the site.xml descriptor:

<project>
  ...
  <skin>
    <groupId>net.ju-n.maven.skins</groupId>
    <artifactId>maven-1024px-skin</artifactId>
    <version>1.0.0</version>
  </skin>
  ...
</project>
